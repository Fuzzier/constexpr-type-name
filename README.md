# Obtain type name at compile time in C++17

It is a tiny header-only library.

Minimum Requirements for Compilers:

| Family | Minimum version | Required flags |
| ------ | --------------- | -------------- |
| GCC    | 8.1.0           | -std=c++17     |
| Clang  | 8.0.0           | -std=c++17     |
| MSVC   | 19.20           | /std:c++17     |
